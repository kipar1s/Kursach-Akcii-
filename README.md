## 📋 Описание проекта

Настольное приложение для покупки и продажи акций с функцией прогнозирования курса методом экстраполяции на основе линейной функции.

Система позволяет:
- Получать актуальные курсы акций и доллара из сети Интернет.
- Хранить данные о пользователях, транзакциях и портфеле в базе данных.
- Прогнозировать динамику курса акций методом линейной экстраполяции.
- Управлять инвестиционным портфелем через графический интерфейс.

## ⚙️ Установка и запуск

### Требования

* Windows 10/11
* Microsoft .NET Framework 4.7.2+ (или .NET 6/8)
* Microsoft Access Database Engine (для работы с .accdb)
* Visual Studio 2019/2022 (для сборки из исходников)

### Сборка и запуск

1. **Клонируйте репозиторий:**
```bash
git clone https://github.com/kipar1s/Kursach-Akcii-.git
```

2. Откройте `WindowsFormsApp1.sln` в Visual Studio.
3. Убедитесь, что файл `TaskActionsBD.accdb` находится в рабочей директории.
4. Нажмите **F5** для сборки и запуска.

## 🗄 База данных

База данных реализована в формате Microsoft Access (`TaskActionsBD.accdb`).  
Структура таблиц описана в файле `SQLQuery1.sql`.

**Основные сущности:**

* **Пользователи** — данные об аккаунтах.
* **Акции** — справочник торгуемых ценных бумаг.
* **Транзакции** — история операций покупки/продажи.
* **Портфель** — текущие позиции пользователя.
* **Курсы валют** — история курсов доллара и акций.

---

## 📡 Получение данных из Интернета

Для получения актуальных котировок используется класс `HttpClient`:

* **Курсы акций** — через открытые API (Yahoo Finance / MOEX ISS / другие).
* **Курс доллара** — через API Центробанка РФ.

Данные загружаются в реальном времени и сохраняются в БД для последующего анализа и прогнозирования.

---

## 📊 Алгоритм прогнозирования

Прогнозирование осуществляется методом линейной экстраполяции на основе линейной функции:  
##`y = k·x + b`

**Этапы работы алгоритма:**

1. Выборка исторических данных курса акции из БД за заданный период.
2. Аппроксимация зависимости методом наименьших квадратов.
3. Вычисление коэффициентов `k` (угловой коэффициент) и `b` (смещение).
4. Экстраполяция значения на заданное количество шагов вперёд.
